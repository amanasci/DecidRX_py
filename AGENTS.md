# AGENTS.md

## Setup commands
- Activate venv: `source .venv/bin/activate`
- Install deps: `pip install -r requirements.txt`
- Run tests: `pytest`


## Tips 
- Always try to run tests.
- For any new functionality, add new test cases and test it.
- Always make sure the code is optimized.
- Always make sure the code is well documented add proper docstrings and type hints.
- Always add any new command to the help command. 
- Cleanup the repo whenever you are done working. Don't leave any unnecesary files.
